Description
--------------------
DirectX9を用いたゲームです。

![screen][url_screen_image]

Features
--------------------
* オリジナル性のあるゲームシステム
* 自作フレームワーク
* FBXからコンバートした独自形式バイナリーデータを使用([FBXConverter1.0][url_fbx_converter])
* Bullet Physics採用
* Doxygen採用

Requirement
--------------------
コンパイルする場合は以下をインストールすること。
* [boost1.69.0][url_boost]

Install
--------------------
ダウンロードしたファイルを展開するだけで使用可能。

Usage
--------------------
キーボード、XInputコントーラー対応
